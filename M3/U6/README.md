*Pinout*
LED RGB:
- RED - GPIO13
- GREEN - GPIO11
- BLUE - GPIO12

BUTTONS (Pull-up interno obrigatório):
- A - GPIO5
- B - GPIO6

MICROPHONE:
- GPIO28 (ADC Channel 2)
- Vmax = 3.3V
- Vmin = 0V
- Vmean = 1.65V
